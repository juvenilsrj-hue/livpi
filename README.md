# LIV Propriedade Intelectual — site

Site institucional + marketplace da **LIV Propriedade Intelectual**: registro de marcas no INPI e
corretagem (compra/venda) de marcas registradas, com pagamento em custódia e transferência de
titularidade averbada no INPI.

## Estrutura

| Arquivo | Página |
|---|---|
| `index.html` | Site único: registro + intermediação de compra e venda de marcas (inclui o marketplace) |
| `liv_mark_dark.png` | Logo (marca) tinta navy, para fundos claros |
| `liv_mark_light.png` | Logo (marca) tinta creme, para fundos escuros |

Site estático, página única (HTML + CSS/JS inline). Abre direto no navegador.

## Deploy

GitHub → **Netlify auto-deploy** (publish directory = raiz). Fazer todos os ajustes localmente e
subir o conjunto completo em um push para a branch `main`.

> Marcas, preços e simulações exibidos nas páginas são **ilustrativos** — substituir pelo catálogo real.
